# 랜덤 이름 뽑기 (정적 웹페이지)

호평지점 선/후배(동료) 이름을 입력해 **랜덤으로 한 명씩 뽑는 웹 게임**입니다.  
별도 빌드 없이 **HTML/CSS/JS 정적 파일 업로드만으로 동작**합니다.

## 기능

- **이름 입력/저장**: 줄바꿈 또는 쉼표로 구분 (예: `양명민, 정선경, 양승호, 양슬지`)
- **랜덤 뽑기 애니메이션**
- **중복 없이 전원 1회씩**(기본값) / **중복 허용** 모드 전환
- **히스토리** + **되돌리기(Undo)**
- 설정/목록/진행상태 **자동 저장(localStorage)**
- 단축키: `Space`(뽑기), `R`(뽑기 초기화)
- 버튼: **링크 복사**

## 사용 방법

1. 이름 입력 → **저장**
2. **뽑기** 버튼 클릭(또는 `Space`)
3. 중복 없이 모드에서 전원이 한 번씩 뽑히면 더 이상 뽑히지 않습니다. 다시 하려면 **뽑기 초기화**를 누르세요.

## 파일 구성

- `index.html`: 화면/문구
- `styles.css`: 디자인
- `app.js`: 뽑기 로직(저장/중복방지/히스토리/효과음)

## 실행 방법(권장)

브라우저에서 `index.html`을 직접 열어도 되지만, 일부 기능(예: 클립보드)이 제한될 수 있어 **로컬 서버 실행**을 권장합니다.

예시(파이썬 설치되어 있을 때):

- PowerShell:
  - `python -m http.server 5173`
  - 브라우저에서 `http://localhost:5173` 접속

## 업로드/배포

이 프로젝트는 정적 파일만 사용합니다. 아래 파일들을 같은 폴더에 업로드하면 됩니다:

- `index.html`
- `styles.css`
- `app.js`

### GitHub Pages (예시)

1. 이 폴더를 GitHub 저장소에 올림
2. 저장소 설정 → Pages → Branch 선택 → Save

### Netlify / Vercel

- “Static site”로 업로드하면 됩니다. 빌드 커맨드 없이, 퍼블리시 디렉토리는 프로젝트 루트로 설정하세요.

## 주의사항(저장/개인정보)

- 이 페이지는 **서버로 데이터를 전송하지 않습니다.**
- 이름/설정/히스토리는 사용 중인 브라우저의 **localStorage(로컬 저장소)**에 저장됩니다.
- 공용 PC에서 사용했다면 **전체 초기화**를 눌러 저장된 데이터를 삭제하세요.
